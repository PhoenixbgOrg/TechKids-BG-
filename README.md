# TechKids-BG — AORUS Academy (PC Hardware Quiz)

**Live demo:** https://phoenixbgorg.github.io/TechKids-BG-/

TechKids-BG is a lightweight, AORUS-themed educational web app designed to help kids (and curious adults) learn the basics of **PC hardware** through short, fun quizzes and quick explanations.

> Tip: Best experience on **desktop or tablet** (bigger screen = better UI/reading).

---

## What you get

- ✅ Kid-friendly quizzes about PC parts and terminology  
- ✅ Clean, AORUS-inspired UI (dark, high-contrast)  
- ✅ Randomized questions (so it doesn’t feel repetitive)  
- ✅ Simple flow: start → answer → feedback → next  
- ✅ Runs fast and works great on low/older machines  
- ✅ Hosted on **GitHub Pages** and deployed via **GitHub Actions**
